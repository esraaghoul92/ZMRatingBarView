# ZMRatingBarView
A simple Rating Bar library by Swift

# Setup

#### Install with [CocoaPods](http://cocoapods.org/)

```ruby
pod 'ZMRatingBarView', '~> 0.1.1'
```

#### [Example](Example/Example/ViewController.swift)

# Configuration

* `score`
* `maxScore`
* `starColor`
* `starSize`
* `starSpaceSize`
